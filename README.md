🚀 Space Waves

A fast-paced, reflex-based arcade game built using React + Canvas, where players navigate through dynamic wave-like tunnels while avoiding obstacles.

🔗 Live Demo:
👉 https://nisaanthsd.github.io/Space_Waves/

🎮 Gameplay

Space Waves is a minimalist arcade game inspired by wave-style mechanics:

Control a moving wave/arrow through a tunnel
Avoid hitting walls and obstacles
Increasing difficulty as you progress
Instant restart for quick retry gameplay
Smooth and responsive controls

The game focuses on precision, timing, and reflex skills.

✨ Features
⚡ Smooth canvas-based rendering
🎯 Simple yet addictive gameplay
🎮 Keyboard controls (Arrow keys / A-D)
🔁 Instant retry system
🌌 Clean dark-themed UI
📱 Responsive design (works on desktop & mobile)
